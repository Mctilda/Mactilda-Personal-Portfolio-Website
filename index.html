Data Warehouse

Project Brief:

This project focuses on designing and implementing a data warehouse solution for a fictional bike store chain, BikeStores, to support enhanced data-driven decision-making. The existing BikeStores database captures daily transactions across different stores, products, and customer details through an Online Transaction Processing (OLTP) system. However, for strategic analysis, it requires a data warehouse to consolidate and analyze historical data.

In this game projecT, we focused on creating a suspenseful atmosphere where anticipation heightened the gameplay.
